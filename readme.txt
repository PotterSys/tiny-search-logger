Logs searches made by end users on the Search form. Use this wisely

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Now all the searches made on your site will appear on Searches menu in your WordPress administration panel
